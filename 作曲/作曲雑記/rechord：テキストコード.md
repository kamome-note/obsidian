rechord は、テキスト入力するだけで実際に演奏もできるコード進行共有サービス。
https://rechord.cc/
![[rechord：テキストコード-1748102471543.png|593x224]]

AIに生成させたコードをとりあえず鳴らしてみるのに便利。

```
# 小節を | で区切ってコードを入力します
C|G|Am7|Em7
F|Em7Am7|Dm7|G7
# 行頭に # を入力するとメモが書けます
FM7|G7
# % で連打、 _ で休符、 = で直前のコードを伸ばします
E7%_Am7|=G
# / でオンコード(最低音)を指定できます
FC/E|Dm7Dm7/G
Cadd9
```
